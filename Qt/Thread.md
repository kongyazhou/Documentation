title: Qt线程

Qt是个全端开发工具。

图形界面运行中不允许阻塞，所以后台程序部分必须在独立于界面的线程中运行。

## 创建线程类

- 新建一个类文件

```
右键工程名，添加新文件。
选择c++，c++ class类。
```

- 新建类的头文件中，应包含"QThread"头文件，并继承于"QThread"类，如: 

```c++
#include <QThread>

class movieThread : public QThread
{
	/*code*/
}
```

- 复写run()

```
通过在线程的源文件中复写run()函数，实现线程的功能。
```

- 线程通信

通过信号与槽的机制实现通信，具体见信号与槽章节。
