# Cordova

```
Cordova提供了一组设备相关的API，通过这组API，移动应用能够以JavaScript访问原生的设备功能，如摄像头、麦克风等。
```



## 常用命令

- 安装 Cordova

	npm install -g cordova

- 创建项目

	cordova create MyApp

- 添加支持的平台

	cordova platform add <platform name>

- 运行你的app

	cordova run <platform name>

- 构建你的app

	cordova build <platform name>
